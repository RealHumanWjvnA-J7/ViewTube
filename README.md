# ViewTube

A lightweight YouTube frontend built with vanilla HTML, CSS, and JavaScript.

Watch videos, search YouTube, load videos directly from URLs, and keep a local watch history all without frameworks.

Try it here:

https://realhumanwjvna-j7.github.io/ViewTube/

## Known Issues

- Some search methods may be blocked by browser CORS policies.
- Public Invidious instances occasionally go offline.
- Direct YouTube search endpoints can change without warning.
